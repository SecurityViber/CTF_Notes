Most common Web Proxies are
- Burp Suit -> Official king... but needs pretty much an enterprise license for everything
- ZAP -> Sicko Stuff for en opensource solution

To make your life easier you should install the firefox extension `Foxy Proxy` which enables you to toggle if you would like to proxy to be on or off.
