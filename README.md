# CTF Notes

Personal knowledge base for CTF challenges, penetration testing, and defensive security (CPTS/CDSA).

## Structure

- **Basics/** — reusable technique and tool references (protocols, tools, privesc, shells, etc.)
- **Certifications/** — notes organized by HTB certification module (CPTS, CDSA)
- **Writeups/** — HTB box write-ups
- **OSINT/** — open-source intelligence resources
- **Home.md** — main entry point for Obsidian navigation

## Getting Started in Obsidian

Open `Home.md` for the full navigation hub, or use the graph view to explore connections between notes.
